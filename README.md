### Module : Introduction à l’informatique quantique et applications
Description générale

Ce module a pour objectif d’introduire les concepts fondamentaux de l’informatique quantique et de présenter leurs applications pratiques à travers des algorithmes et des cas d’usage concrets. Les séances s’appuient sur des notebooks Python utilisant la librairie Qiskit pour la conception, la simulation et l’analyse d’algorithmes quantiques.

Le module est structuré en trois séances progressives, allant des bases du calcul quantique jusqu’aux problématiques d’optimisation et de Quantum Machine Learning (QML).

# Prérequis techniques

Les notebooks fournis ont été développés et testés avec les versions suivantes :

Python : 3.13
⚠️ Ne fonctionne pas avec Python 3.14

qiskit : 2.2.3

qiskit_aer : 0.17.2 (utilisé pour la simulation)

Important
Les librairies Qiskit évoluent rapidement. Il est fortement recommandé d’utiliser les versions exactes indiquées ci-dessus afin d’éviter des erreurs de compatibilité lors de l’exécution des notebooks.

## Organisation du dépôt
# Séance 1 – Introduction à l’informatique quantique

Cette séance vise à introduire :
Les concepts fondamentaux du module
Les principes de base du codage et de l’exécution d’algorithmes quantiques
L’utilisation de la librairie Python Qiskit



# Séance 2 – Circuits variationnels et application au Quantum Machine Learning

Cette séance est consacrée à l’implémentation de circuits variationnels appliqués au Quantum Machine Learning.
Les points abordés :
Résolution classique d’un problème de classification simple
Deux approches quantiques pour le même problème
Simulation des expériences et estimation des performances


# Séance 3 – Problèmes d’optimisation et défis des applications quantiques

Cette séance applique l’informatique quantique à un problème d’optimisation à l’aide de l’algorithme QAOA (Quantum Approximate Optimization Algorithm).
Elle inclut également :
Une rétrospective sur l’ensemble du module
Une discussion sur les promesses et les limites actuelles de l’informatique quantique


## Remarques finales

Tous les calculs quantiques sont réalisés en simulation.
L’utilisation d’un backend réel est uniquement illustrée via ses caractéristiques techniques.
